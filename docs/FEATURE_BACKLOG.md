# Feature Backlog

This is the working backlog for future Spotlite slices. Keep it practical and presenter-focused.

## High-Value Next Features

- Typing Focus style and fallback tuning
- Terminal-specific Typing Focus support for apps like Ghostty when caret geometry is not exposed through Accessibility
- Timer placement and end-state customization

## Permission-Gated Features

- True zoom and magnifier through ScreenCaptureKit
- Screenshot and region snip
- OCR from selected region
- Screen recording
- Keystroke display
- DemoType-style scripted typing

## Integration Ideas

- macOS Shortcuts actions
- Stream Deck integration hooks
- Import/export profiles for teaching, live coding, and screen recording
- Community-contributed shortcut presets
