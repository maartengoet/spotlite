# Spotlite Roadmap

Spotlite is built as a free, MIT-licensed macOS presenter tool. The roadmap should stay practical: every feature should make live presenting, teaching, screen sharing, or recording easier without adding friction during a talk.

## Milestone 1: Presenter Overlay Foundation

- Menu bar app lifecycle
- Global shortcuts
- Multi-display transparent overlay
- Draw mode without zoom
- Clear and undo annotations
- Permissions guide
- About panel
- Keyboard shortcuts guide
- Auto Start at Login setting
- App bundle build script

## Milestone 2: Annotation Tools

- Pen and highlighter tools
- Color palette
- Stroke width controls
- Straight line, arrow, rectangle, ellipse
- Whiteboard and blackboard modes
- Text annotations
- Auto-erase option

## Milestone 3: Attention Tools

- Typing Focus for the active text caret
- Spotlight/focus dimming
- Adjustable spotlight size
- Magnifying lens
- True live zoom
- Click indicators
- Dim inactive windows
- Hide/blur sensitive region

## Milestone 4: Presenter Workflow

- Countdown timer and stopwatch
- Presenter notes overlay
- Keystroke display for tutorials
- Profiles for live coding, teaching, sales demos, and recording
- Stream Deck and macOS Shortcuts integration

## Milestone 5: Capture Helpers

- Screenshot and region snip
- Copy to clipboard or save to file
- Screen recording through ScreenCaptureKit
- Region/window recording
- OCR from selected region

## Milestone 6: Demo Automation

- DemoType-style scripted typing
- Snippet queue
- User-driven typing mode
- Safe clipboard import with explicit marker
- Pause and key-control commands
